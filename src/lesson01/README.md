# Lesson 01 - Class & Object

## 📖 Overview

This lesson introduces the two most fundamental concepts of Object-Oriented Programming (OOP): **Class** and **Object**.

## 🎯 Objectives

- Understand what a class is
- Understand what an object is
- Create objects from classes
- Use constructors
- Call methods

## 📂 Examples

- Student
- Car
- Book
- Employee
