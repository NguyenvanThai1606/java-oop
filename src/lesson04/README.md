# Lesson 04 - Inheritance

## 📖 Overview

This lesson introduces inheritance in Java.

## 🎯 Objectives

- Understand inheritance
- Reuse code with `extends`
- Initialize parent classes using `super`
- Build a simple class hierarchy

## Class Diagram

```text
Person
│
├── Student
└── Teacher
```

## Key Concepts

- Inheritance
- extends
- super
- Code Reuse

## What I Learned

- Child classes inherit fields and methods from parent classes.
- `super()` is used to call the parent constructor.
