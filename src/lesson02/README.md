# Lesson 02 - Constructor

## 📖 Overview

This lesson focuses on constructors and constructor overloading in Java.

## 🎯 Objectives

- Understand constructors
- Create multiple constructors
- Learn constructor overloading
- Initialize objects correctly

## 📂 Example

Student class with:

- Default constructor
- Constructor with two parameters
- Constructor with three parameters

## ▶️ Expected Output

```text
Default Constructor

ID   : SV001
Name : Thai Nguyen
Age  : 0

ID   : SV002
Name : John
Age  : 20
```

## 🧠 Key Concepts

- Constructor
- Default Constructor
- Parameterized Constructor
- Constructor Overloading

## 📌 What I Learned

- Constructors initialize objects
- Multiple constructors improve flexibility
- Java supports constructor overloading
